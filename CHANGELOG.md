# BiliVR-Max 更新日志

## [v1.1] - 2026-04-03
### 修复与优化
1. **字幕视口对齐 (Viewport Tracking)**
   - 改写了字幕渲染逻辑，字幕现在会动态跟踪每个眼睛的实际渲染视口 (Viewport)，不再是生硬的平分全屏。
   - 适配了当修改“画面大小 (Scale)”和“画面间距 (Gap)”时，字幕位置的同步变动。
2. **立体深度视差优化**
   - 将字幕深度计算公式改为基于视口宽度的比例视差，极大增强了不同分辨率下的立体适应性。
   - 修复了当深度为 0 时，字幕会重叠在全屏中央的问题。
3. **新增功能**
   - 在 VR 设置面板中新增了“字幕大小”调节选项。用户可以在 2.0% - 10.0% 的视觉范围内自由调节字体大小。
4. **权限说明 (User Fixed)**
   - 解决了本地字幕加载所需的存储读取权限问题。
